#  React Calculator App

A simple, responsive calculator application built using **React JS** and **Bootstrap 5**.  
This project demonstrates core React concepts, clean UI design, and basic arithmetic logic.

---

## Description

The **React Calculator App** allows users to perform basic mathematical operations such as addition, subtraction, multiplication, and division.  
It is designed with a clean and minimal interface using Bootstrap 5 and is fully responsive across devices.

This project is ideal for:
- Learning React fundamentals
- Practicing component-based architecture
- Showcasing a simple but functional UI project in a portfolio

---

##  Features

-  Addition
-  Subtraction
-  Multiplication
-  Division
-  Instant calculation results
-  Responsive design (Bootstrap 5)
-  Clean and user-friendly UI

---

##  Technologies / Libraries Used

- **React JS** – Frontend library
- **Bootstrap 5** – UI & responsive layout
- **JavaScript (ES6+)**
- **HTML5**
- **CSS3**
- **Node.js & npm** – Package management
